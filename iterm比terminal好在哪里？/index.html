iterm比terminal好在哪里？

为什么要用iterm

  1. 多窗口切换
command + T
  1. 同个窗口横向分屏
command + d
  1. 同个窗口，竖向分屏
command + shift + d
  1. 自动补齐命令
comm + ;
  1. 直接打开文件
按住command键并点击文件名 (ps: terminal用open命令也可以)
  1. 自动复制
直接双击需要复制的文字，即可自动复制
  1. 设置主题配色
在偏好设置里进行一些基本的颜色设置，在GitHub上下载做好的主题，导入。

(ps: 成品自由度高👍，合心意的需要手动配置->terminal也可以手动配置)

这么好的东西我还是用terminal好了。
