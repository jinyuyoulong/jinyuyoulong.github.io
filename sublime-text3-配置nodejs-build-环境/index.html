sublime text3 配置nodejs build 环境

March 3, 2017 node Sublime Text

可参考使用Sublime Text 3 进行Python3开发

Tools–>Build System–>New Build System 写入

{
    "shell_cmd": "/usr/local/bin/node ${file}",
    "selector": "source.javascript",
    "file_regex": "^(...*?):([0-9]*):?([0-9]*)",
    "working_dir": "${file_path}",
}

保存为/Packages/User/node.sublime-build

然后选择环境为node即可
