# 消息队列 MQ

能随便讲个应用场景嘛

发短信，发邮件

大量的定时任务

支付回调也经常用

你付款之后有很多步骤

主要是付款是异步过程。

你打钱，支付宝服务器通知电商后台服务器

然后电商后台接到请求后，发送一个消息去队列里，带上订单信息

然后服务器集群里有人接活
