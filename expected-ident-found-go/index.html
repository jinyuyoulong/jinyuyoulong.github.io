Go 编译报错 expected 'IDENT', found 'go'

报错 expected ‘IDENT’, found ‘go’

运行go run

文件目录为

./go-demo/main.go

由于package 命名为go-demo，命名格式不规范，导致的这个问题。

解决办法：将go-demo 统一改为 gostart ，去掉 - 字符 报错解决。

反思：包的名字应只用小写。不要用下划线式，也不要用驼峰式。使用单数
